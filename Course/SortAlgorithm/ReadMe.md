# 7.排序

- 概念
  - 将各元素按关键字递增/或递减顺序重新排列
- 评价指标
  - 稳定性关键字相同的元素经过排序后相对顺序是否会改变
  - 时间复杂度、空间复杂度
- 分类
  - 内部排序 
    - 数据都在内存中
  - 外部排序 
    - 数据太多，无法全部放入内存
